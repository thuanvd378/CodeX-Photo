# CodeX Photo Demo Script

## 2-Minute Flow

1. Open the app with `npm run dev`.
2. Click **Tải ảnh lên**.
3. Select a product photo in PNG, JPG, or WebP.
4. Click **Mẫu Shopee** or type the prompt below.
5. Click **Tạo ảnh**.
6. Show the AI edit plan panel before focusing on the preview.
7. Toggle **Trước**, **Sau**, and **So sánh**.
8. Open **Cài đặt** and show API status, demo mode, theme, accent, and privacy text.
9. Choose **Shopee 1:1 - 1024x1024** in export settings.
10. Click **Xuất**.
11. Click an older thumbnail in **Lịch sử** to restore a previous result.

## Vietnamese Prompts

```text
Đổi nền thành trắng sạch như ảnh Shopee, làm sản phẩm sáng hơn, giữ nguyên logo và hình dáng.
```

```text
Làm ảnh sắc nét hơn, tăng sáng nhẹ, giữ nguyên chữ trên bao bì.
```

```text
Tạo ảnh TikTok Shop 4:5, nền trắng, sản phẩm ở giữa, giữ nguyên màu thật và logo.
```

## What To Show

- The result is labeled as demo mode when no API key is used.
- The plan lists internal tool commands before execution.
- Export downloads PNG/JPG/WebP using marketplace presets.
- API key status is masked after saving.
