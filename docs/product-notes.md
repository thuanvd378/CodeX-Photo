# Product Notes

## First Customer

The first customer is a Vietnamese online seller who needs clean product images for Shopee, TikTok Shop, Facebook, and short-form commerce listings.

## Pain Point

Small sellers often do not know Photoshop, do not have design staff, and cannot spend time manually selecting products, masking backgrounds, preserving logos, resizing canvases, and exporting marketplace-specific ratios.

## Why AI-Controlled Editing Matters

The user should describe the commercial outcome, not operate technical tools. CodeX Photo turns a plain-language instruction into an auditable edit plan, then executes the internal tools in order.

## Difference From Photoshop

Photoshop exposes tools first and requires the user to know how to operate them. CodeX Photo exposes intent first. Layers, masks, adjustments, presets, and export decisions exist internally, but the AI agent acts as the operator.

## Path To A Serious Product

- Add robust subject segmentation and logo/text preservation checks.
- Integrate model-specific image edit APIs for inpainting and background generation.
- Add batch processing for seller catalogs.
- Add marketplace compliance presets.
- Add local project files with non-destructive edit history.
- Package signed installers for Windows and macOS.
